# myTomori 启动说明

本项目已提供两个基于 `ReactAgent` 的对话前端：

1. `chat_streamlit.py`（Web 页面）
2. `chat_terminal.py`（终端交互）

## 1. 安装依赖

在项目根目录执行：

```bash
uv sync
```

## 2. 配置环境变量

在 `.env` 中至少配置：

```env
MODEL=你的模型名
DASHSCOPE_API_KEY=你的API_KEY
BASE_URL=你的模型服务地址
bangumi_acg_agent_token=你的bangumi访问令牌
```

## 3. 启动 Streamlit 前端

```bash
uv run streamlit run chat_streamlit.py
```

启动后，浏览器会自动打开对话页面。

## 4. 启动终端前端

```bash
uv run python chat_terminal.py
```

输入问题即可开始对话，输入 `exit` / `quit` / `退出` 可结束。
